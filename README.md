# 🚀 @imksh Packages

A collection of reusable npm packages built and maintained by **Karan Kumar (@imksh)**.

These packages are designed to speed up development by providing production-ready templates, utilities, UI components, CLI tools, and developer resources.

---

## 📦 Packages

| Package | Description | Status |
|---------|-------------|--------|
| `@imksh/create` | Create React, Node, MERN and Next.js projects from production-ready templates. | 🚧 In Development |
| `@imksh/ui` | Reusable React UI components built with Tailwind CSS & DaisyUI. | Planned |
| `@imksh/icons` | Custom SVG icon library. | Planned |
| `@imksh/utils` | Common JavaScript and Node.js utility functions. | Planned |
| `@imksh/config` | Shared ESLint, Prettier and configuration packages. | Planned |

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/imksh/npm-packages.git
```

Install dependencies

```bash
npm install
```

---

## 📦 Publishing

Navigate to the package directory

```bash
cd create
```

Update version

```bash
npm version patch
```

Publish

```bash
npm publish --access public
```
